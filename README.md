## ✨ 特性

- Monorepo 管理模式：`cdk`, `components`, `pro`
- 全面拥抱 Composition Api，从源码到文档
- 完全使用 TypeScript 开发，提供完整的类型定义
- 开箱即用的 Tree Shaking
- 高覆盖率的单元测试
- 国际化语言支持
- 灵活的全局配置
- 深入细节的主题定制能力

## 🖥 支持环境

- Vue `^3.0.0` [![vue-next](https://img.shields.io/npm/v/vue/next.svg)](https://www.npmjs.com/package/vue/v/next)
