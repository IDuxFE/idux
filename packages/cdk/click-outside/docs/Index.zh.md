为除了特定元素外添加全局点击事件。

## 何时使用

- 如点击浮层外部关闭。

## API

### ClickOutside

#### ClickOutsideOptions

```typescript
interface ClickOutsideOptions {
  exclude: Ref<HTMLElement>[]
  handler: ClickOutsideHandler
}

type ClickOutsideHandler = () => void

type ClickOutsideBinding = ClickOutsideHandler | ClickOutsideOptions
```
