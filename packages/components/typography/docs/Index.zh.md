文本的基本格式。

## 何时使用

- 当需要展示标题、段落、列表内容时使用，如文章/博客/日志的文本样式。
- 当需要一列基于文本的基础操作时，如拷贝/省略/可编辑。

## API

### `ix-typography`

```typescript
type TypographyType = 'success' | 'warning' | 'error' | 'secondary'

interface TypographyOptions {
  type?: TypographyType
  disabled?: boolean
}

type TypographyConfig = TypographyType | TypographyOptions
```
