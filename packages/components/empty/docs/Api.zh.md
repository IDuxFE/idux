#### EmptyProps

| 名称 | 说明 | 类型  | 默认值 | 全局配置 | 备注 |
| --- | --- | --- | --- | --- | --- |
| `description` | 自定义描述内容 | `string` | - | - | - |
| `image` | 自定义图片 | `string \| VNode` | - | ✅ | - |
| `simple` | 简易模式 | `boolean` | - | - | - |
